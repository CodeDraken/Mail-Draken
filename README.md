# Mail-Draken
Mail, surveys, and feedback MERN app.

You can view it here :
https://cryptic-gorge-73107.herokuapp.com/

This is a sample app made for learning.

The tech stack includes ( but not limited to ):
* Back End
  * NodeJs
  * Express
  * body-parser
  * Lodash
  * SendGrid
  * Stripe
  * Passport
  * passport-github
  * passport-google-oauth20
  * path-parser
  
* Front End
  * Axios
  * Babel
  * Materialize
  * React
  * Redux
  * redux-form
  * redux-thunk
  
* Dev Tools
  * create-react-app
  * nodemon
  * concurrently
  * localtunnel
